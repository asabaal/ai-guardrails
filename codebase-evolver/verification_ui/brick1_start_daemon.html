What this function does:

start_daemon(repo_path: str, port: int) -> PyrightProcess

This function starts the Pyright language server as a daemon process. It:

  • Validates that repo_path is absolute and exists
  • Checks that the port is in valid range (1024-65535)
  • Ensures the port is not already in use
  • Starts Pyright with detached stdout/stderr
  • Returns a PyrightProcess object with process handle and metadata
