What this function does:

stop_daemon(process: PyrightProcess) -> None

This function stops a running Pyright daemon process. It:

  • Validates that process is a PyrightProcess object
  • Checks if process is already stopped
  • Attempts graceful termination (SIGTERM)
  • Waits up to 5 seconds for graceful shutdown
  • Force kills (SIGKILL) if graceful fails
  • Guarantees cleanup even on errors
